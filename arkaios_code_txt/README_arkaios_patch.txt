# ARKAIOS patch (mini)
Archivos:
- `arkios-core.html` — UI con registro del worker, diagnóstico en vivo y botones “Ver imagen adjunta / Visitar sitio web”.
- **Requiere**: `arkaios-worker.js` en la misma carpeta (el tuyo actual).

Uso:
1) Sube `arkios-core.html` y `arkaios-worker.js` a la misma ruta de tu hosting (https recomendado).
2) Abre `arkios-core.html`. Verás el estado de Puter/IA/Imágenes. Si Puter está activo, podrás chatear, subir y generar imágenes.
3) Para generar imágenes escribe: `img: una casa futurista`.
