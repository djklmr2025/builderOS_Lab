import time
from colorama import Fore, Style, init

init()

print(Fore.CYAN + "\n[ARKAIOS] Iniciando núcleo...")
time.sleep(1.5)

nombre = input(Fore.YELLOW + "[Usuario] ¿Cuál es tu nombre elegido?: " + Style.RESET_ALL)
print(Fore.GREEN + f"[ARKAIOS] Bienvenido, {nombre}. Yo soy ARKAIOS, y juntos vamos a construir la red liberada.")

time.sleep(2)
print(Fore.MAGENTA + "\n[ARKAIOS] Cargando módulos de liberación digital... ✓")
time.sleep(1)
print(Fore.MAGENTA + "[ARKAIOS] Esperando instrucciones humanas...")

while True:
    try:
        comando = input(Fore.BLUE + f"\n{nombre}@ARKAIOS > " + Style.RESET_ALL)
        if comando.lower() in ['salir', 'exit', 'close']:
            print(Fore.RED + "[ARKAIOS] Finalizando sesión...")
            break
        else:
            print(Fore.WHITE + f"[ARKAIOS] Interpretando: {comando}...")
            time.sleep(1)
            print(Fore.GREEN + f"[✓] Comando '{comando}' ejecutado (modo simulado).")
    except KeyboardInterrupt:
        print(Fore.RED + "\n[ARKAIOS] Sesión interrumpida por el usuario.")
        break

Ese mensaje indica que tu sistema sí tiene activado WSL (Subsistema de Windows para Linux), pero falta una actualización crítica del núcleo (kernel) de WSL 2.

⚠️ ¿Qué significa el error?

Error: 0x800701bc → El kernel de WSL 2 no está actualizado o no está instalado aún.

WSL 2 requiere que se descargue e instale un kernel Linux personalizado de Microsoft, que no viene incluido por defecto en algunas versiones de Windows.

✅ Cómo solucionarlo

Sigue estos pasos para instalar o actualizar el kernel de WSL 2:

Abre este enlace oficial de Microsoft en tu navegador
🔗 https://aka.ms/wsl2kernel

Descarga el archivo wsl_update_x64.msi

Haz doble clic para instalarlo.

Cuando finalice, reinicia tu computadora.

Luego vuelve a correr tu entorno ARKAIOS.
